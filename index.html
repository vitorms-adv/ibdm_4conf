Fonte de Dados — Google Sheets

A URL mestra (CSV_URL) está definida no código. Informe o gid de cada aba. O gid aparece na URL do Sheets ao clicar na aba: …/edit#gid=XXXXXXXX

Atualização Automática

O valor padrão (AUTO_REFRESH_MS) também pode ser editado no código.

Como publicar no Google Sites

1. Na planilha: Arquivo → Publicar na web → Planilha inteira → CSV → Publicar.
2. Copie a URL e cole como CSV_URL no código.
3. Para cada aba, anote o gid= da URL e preencha ao lado.
4. No Google Sites: Inserir → Incorporar → Cole o HTML completo.
5. Clique em Atualizar no header para puxar os dados imediatamente.
